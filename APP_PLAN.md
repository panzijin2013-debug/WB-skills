# 双击本地 App 方案

本文档说明如何把当前命令行工具升级为普通用户可用的本地 App。

## 当前实现状态

已经完成 macOS Electron MVP：

- App 可以双击启动。
- App 可以选择本地归档文件夹。
- App 可以初始化标准归档目录。
- App 可以打开微博登录窗口。
- App 可以检测登录后的微博 UID。
- App 可以通过登录态读取自己的微博时间线。
- App 可以下载微博图片到本地 `media/`。
- App 可以校验已有 `data/posts.json` 和 `media/`。
- App 可以生成 `html/archive.html`。
- App 可以打开生成的本地网页和归档文件夹。

尚未完成：

- 断点续采集。
- 缺失图片重试下载。
- 暂停/继续采集。
- 正式签名、公证和 Windows 打包。

## 产品定位

这是一个个人使用的本地微博归档工具，不做云端服务，不做商业化收费，不要求用户安装开发环境。

核心体验：

```text
下载压缩包 -> 解压 -> 双击 App -> 登录微博 -> 点击开始备份 -> 打开本地归档
```

## 技术形态

建议使用桌面 App 包装当前底层能力。

可选方案：

- Electron：实现最快，容易集成 Node、Playwright、文件系统和内置浏览器。
- Tauri：包体更小，但浏览器自动化和 Node 工具链整合成本更高。

当前阶段建议先用 Electron 做 MVP，因为采集微博需要浏览器控制、登录态管理、文件读写和进度展示，Electron 的集成成本最低。

## App 内部模块

```text
Desktop App
  UI 界面
    欢迎页
    保存目录选择
    微博登录页
    备份范围选择
    进度页
    校验结果页
    完成页

  Core
    collector：微博采集
    downloader：图片下载
    verifier：完整性校验
    renderer：离线页面生成
    storage：断点、日志、配置
```

当前项目里的 `collector`、`verify` 和 `render` 已经接入桌面 UI。下一步要补的是断点续采集、失败重试和更完整的异常处理。

## 用户界面流程

### 1. 欢迎页

显示一句话：

```text
把你的旧微博保存到本地，离线浏览，长期保留。
```

主要按钮：

```text
开始
```

### 2. 保存目录页

按钮：

```text
选择保存位置
```

选择后显示：

```text
保存到：/Users/xxx/Documents/我的微博归档
```

### 3. 微博登录页

按钮：

```text
打开微博登录
```

用户在内置浏览器中自己登录。

登录成功后显示：

```text
微博已登录
```

### 4. 备份范围页

选项：

- 全部微博。
- 指定年份。
- 从上次中断位置继续。

默认选中：

```text
全部微博
```

### 5. 备份进度页

显示：

- 已扫描页数。
- 已保存微博数量。
- 已保存图片数量。
- 当前年份或当前时间位置。
- 缺失图片数量。

按钮：

- 暂停。
- 继续。
- 停止。

### 6. 校验结果页

显示：

```text
微博：1080 条
图片：400 张
缺失图片：0
重复记录：0
```

如果有问题，显示：

```text
重试缺失图片
导出问题清单
```

### 7. 完成页

按钮：

```text
打开我的微博归档
打开保存文件夹
```

## 数据目录

App 应生成下面结构：

```text
我的微博归档/
  archive.config.json
  data/
    posts.json
  media/
  html/
    archive.html
  logs/
    collect.log
    verify.log
  tmp/
    checkpoint.json
```

## 隐私原则

App 必须遵守：

- 不保存微博密码。
- 不上传微博文字和图片。
- 不采集他人主页。
- 不默认导出评论区里的他人隐私信息。
- 所有生成文件都放在用户选择的目录。

## MVP 验收标准

MVP 需要满足：

- 用户双击 App 可以启动。
- 用户可以选择保存目录。
- 用户可以在 App 内登录微博。
- App 可以采集至少一个真实账号的历史微博。
- App 支持中断后继续。
- App 可以下载微博图片。
- App 可以校验缺失图片。
- App 可以生成离线 HTML 页面。
- 断网后 HTML 页面仍能打开并搜索。
